Site Estático com AWS S3 + CloudFront

Objetivo deste HTML

Este arquivo HTML foi criado para demonstrar o funcionamento de um site estático hospedado na AWS, utilizando o serviço S3 como armazenamento e CloudFront como CDN para distribuir o conteúdo com alta performance e baixa latência.

```

Por que este projeto importa?

Ele mostra como é possível colocar um site no ar sem usar servidores, com segurança, escalabilidade e custo quase zero. Tudo isso com uma arquitetura limpa, bem documentada e com foco em boas práticas.

Sobre o conteúdo

Este é um exemplo simples, mas feito com atenção aos detalhes. O estilo foi aplicado diretamente neste arquivo para manter a proposta de site leve e fácil de manter.
```
